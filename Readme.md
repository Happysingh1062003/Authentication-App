# üîê Full-Stack MERN Authentication System

A complete, secure authentication solution built with the MERN stack. This project implements advanced features like **Email Verification**, **OTP-based Password Reset**, and **Persistent Sessions** using JWT and HTTP-only cookies.

---

## üåü Key Features

* **User Registration & Login**: Secure account creation and authentication.
* **Email Verification**: Ensures users provide valid email addresses via a 6-digit OTP.
* **Password Recovery**: Multi-step "Forgot Password" flow (Email Check -> OTP Verification -> Password Reset).
* **Automatic Focus OTP**: Smart UI that automatically moves the cursor to the next input field as you type.
* **Secure Logouts**: Server-side cookie clearing to ensure session termination.
* **Global State Management**: Powered by React Context API for seamless user data access across the app.
* **Responsive Design**: Modern UI built with Tailwind CSS.
